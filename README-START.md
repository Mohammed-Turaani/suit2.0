# Suti 1.9 — 3D Suit Designer (Ready)

## تشغيل الواجهة (React + Vite)

```bash
cd frontend
npm i
npm run dev
```

## المميزات التي تم تجهيزها
- زر **تشغيل/إيقاف الدوران التلقائي** للمجسم (TopBar)
- زر **Reset** لإرجاع زاوية الكاميرا
- Presets جاهزة: **Classic Slim / Regular / Tuxedo / Modern / Full Pack**
- تغيير **لون القماش** للقطعة المحددة (Color Picker + Swatches) + زر تطبيق نفس اللون على كل القطع
- تحكم الكاميرا:
  - بالماوس: سحب للدوران + عجلة للزووم
  - بالأزرار على الشاشة + لوحة المفاتيح

## الموديل والملابس
المشروع يستخدم `public/models/human.glb` وفيه قطع بدلة (Jacket/Pants/Shirt/Vest/Tie) ويمكن إظهار/إخفاء كل قطعة من الشريط العلوي.

> إذا بدك أضيف موديل رجل بدقة أعلى أو قطع ملابس GLB منفصلة (jacket.glb…)، ابعتلي الملفات وسأربطها لك فوراً.
